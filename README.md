# Personal Static Site
Website URL: https://l-web666.github.io

## Project Overview
This is a free static personal website hosted on GitHub Pages, built with pure native HTML. It has no back-end services, no advertisements, and is 100% free to use. All pages run locally in your browser; no user data will be collected or uploaded to any server.

## Repository File Structure
- index.html：The only homepage & entry of the website
- README.md：Project documentation

## Usage Instructions
1. Online access: Visit https://l-web666.github.io directly
2. Offline local preview: Download all files from this repository, then double-click `index.html` to open the website locally.
