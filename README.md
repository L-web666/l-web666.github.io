# 个人静态站点
站点访问地址：https://l-web666.github.io

## 项目简介
基于 GitHub Pages 搭建的免费静态个人站点，纯原生 HTML 开发，无后端、无广告、完全免费。
所有页面仅在浏览器本地加载，不收集、不上传任何用户数据。

## 仓库文件结构
- index.html：网站唯一首页，站点访问入口
- README.md：项目说明文档
## 使用说明
1. 线上访问：直接打开 https://l-web666.github.io
2. 本地离线打开：下载仓库全部文件，双击 `index.html` 即可本地预览页面。

## 开源规范
项目代码完全开源，仅允许个人学习、非商用自用；
禁止倒卖、二次封装商用、违规批量爬虫等用途。
